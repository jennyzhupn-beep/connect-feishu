周四 7月9日
🥘 午餐

腊肠土豆焖饭

🍚 电饭煲 · 3分钟备料
  1. 大米半杯（60g）洗净入锅，加水至刚没过米面
  2. 土豆半个切小丁，干香菇2朵泡发切丁，腊肠半根切片
  3. 所有食材铺在米上，淋生抽1勺+蚝油半勺+香油，不要搅拌
  4. 煮饭键，跳闸后焖5分钟，撒葱花拌匀

🌙 晚餐

水蒸蛋 + 蒜蓉金针菇

🔥 微波炉 · 5分钟
  1. 蛋1个打散+1.5倍温水+少许盐，盖保鲜膜扎孔
  2. 金针菇一小把去根撕散，铺碗底淋蒜蓉酱
  3. 两个碗同时入微波炉，中高火3-4分钟
  4. 蒸蛋淋生抽香油，开吃

周五 7月10日
🥘 午餐

一颗番茄饭

🍚 电饭煲 · 5分钟备料
  1. 大米半杯洗净入锅，水量比平时略少
  2. 番茄1个去蒂划十字放中间，周围铺土豆丁、胡萝卜丁、玉米青豆、火腿丁
  3. 淋生抽1勺、蚝油半勺、黑胡椒少许，煮饭键
  4. 跳闸后压碎番茄，拌匀，焖2分钟
📺 一只番茄饭做法B站
🌙 晚餐

脆皮椒盐鸡翅 + 烤西兰花

🍗 空气炸锅 · 20分钟
  1. 鸡翅4个划刀，料酒+生抽+椒盐腌15分钟
  2. 鸡翅入锅，180℃ 18分钟，中途翻面
  3. 最后5分钟放入西兰花小朵（提前拌油盐）一起烤

周六 7月11日
🥘 午餐

番茄鸡蛋焖面

🍳 小电锅 · 10分钟
  1. 番茄1个切块，小电锅倒少许油炒出沙
  2. 加一碗水烧开，放一小把挂面（约80g）
  3. 面熟后淋入蛋液1个，加生抽1勺+盐+香油
  4. 撒葱花出锅，直接端锅吃
📺 番茄焖面教程B站
🌙 晚餐

咖喱鸡肉焖饭

🍚 电饭煲 · 5分钟备料
  1. 鸡胸肉小半块切丁，料酒+生抽腌10分钟
  2. 米半杯入锅，铺鸡丁+土豆丁+胡萝卜丁
  3. 咖喱块切一半（约15g）放在上面，加水至正常煮饭线
  4. 煮饭键，跳闸拌匀

周日 7月12日
🥘 午餐

葱香嫩滑鸡腿饭

🍚 电饭煲 · 10分钟备料
  1. 鸡全腿1只去骨，料酒+黑胡椒+盐腌15分钟
  2. 电饭煲倒少许油，鸡皮面朝下煎至金黄
  3. 取出鸡腿，放入半杯米+水，铺上葱段、姜片、洋葱丝
  4. 鸡腿放在最上面，煮饭键煮好，切块淋生抽

🌙 晚餐

虾仁豆腐焖蛋

🔥 微波炉 · 7分钟
  1. 嫩豆腐半盒切块铺碗底，速冻虾仁5-6只撒上面
  2. 鸡蛋1个打散+半勺生抽搅匀，淋入碗中
  3. 盖保鲜膜扎孔，高火5分钟
  4. 撒葱花+少许蒸鱼豉油

周一 7月13日
🥘 午餐

清汤菌菇面

🍳 小电锅 · 10分钟
  1. 小电锅加水烧开，放干香菇2朵+白玉菇少许煮3分钟
  2. 下挂面一小把（约80g），加生抽1勺+盐
  3. 面将熟时打入鸡蛋1个，放几片青菜
  4. 滴香油出锅，暖胃又简单
📺 清汤面教程B站
🌙 晚餐

孜然金针菇 + 烤豆腐

🍗 空气炸锅 · 15分钟
  1. 老豆腐半块切厚片，金针菇一小把去根撕散
  2. 豆腐和金针菇刷薄油，撒孜然粉+辣椒粉+盐
  3. 豆腐入锅180℃ 12分钟，中间翻面；金针菇最后8分钟放入
  4. 出锅撒香菜/葱花

电饭煲三汁焖锅

🍚 电饭煲 · 10分钟备料
  1. 鸡翅3个划刀，料酒+生抽腌10分钟
  2. 锅底刷油，铺洋葱丝+土豆块+胡萝卜块+金针菇
  3. 最上面放鸡翅，淋酱汁（生抽1勺+蚝油1勺+番茄酱1勺+糖少许+水3勺）
  4. 煮饭键约40分钟，开盖香气四溢

微波炉番茄鸡蛋面

🔥 微波炉 · 3分钟
  1. 挂面一小把入碗加水没过，高火2分钟
  2. 取出倒掉一半水，加入番茄块+蛋液+生抽+香油
  3. 高火再叮1分钟，拌匀开吃
  4. 一个碗搞定，零洗锅

🥘 午餐

微波炉煲仔饭

🔥 微波炉 · 12分钟
  1. 米半杯+水（米水1:1.2）入微波碗，高火6分钟
  2. 取出铺腊肠片+香菇片+焯水青菜，打一个蛋
  3. 盖盖高火再叮4分钟，焖2分钟
  4. 淋酱汁（生抽1勺+蚝油半勺+糖少许+水1勺）

日式小火锅

🍳 小电锅 · 15分钟
  1. 小电锅加水+味噌半勺（或酱油+盐底）烧开
  2. 放入豆腐块+金针菇+娃娃菜几片+肥牛卷4-5片
  3. 慢慢涮着吃，蘸生抽+蒜末
  4. 最后汤里下几根面条收尾
